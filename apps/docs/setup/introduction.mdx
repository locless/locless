## Prerequisites

- Created your [Locless account](https://locless.com/auth/sign-up)
- Created a project in the [Locless dashboard](https://locless.com/app)

<Steps titleSize="h3">

<Step title="Install Locless CLI">
Install Locless CLI

<Tabs>
  <Tab title="npm">
  ```bash
  npm install -g @locless/cli
  ```
  </Tab>
  <Tab title="pnpm">
  ```bash
  pnpm install -g @locless/cli
  ```
  </Tab>
  <Tab title="yarn">
  ```bash
  yarn global add @locless/cli
  ```
  </Tab>
  <Tab title="bun">
  ```bash
  bun install -g @locless/cli
  ```
  </Tab>
</Tabs>
</Step>

<Step title="Create Locless folder">
Create `locless` folder in your project root

```bash
mkdir locless
```

</Step>

<Step title="Add API keys">
Go to [Locless dashboard](https://locless.com/app) and copy your API keys inside Project settings.
Then add them to `.env` file.

Private key should be always `LOCLESS_AUTH_KEY` but you can use any name for public key.

```bash /.env
EXPO_PUBLIC_LOCLESS_PUBLIC_KEY=<PUBLIC_KEY>
LOCLESS_AUTH_KEY=<AUTH_KEY>
```
</Step>

</Steps>
